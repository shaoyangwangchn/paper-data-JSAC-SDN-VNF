# paper_data

## Detailed meaning

### COST266_data_t.xlsx
**Row**  A: Serial number of nodes  

         B: Abscissa of nodes  
        
         C: Ordinate of nodes
        
         E: Serial number of links
        
         F: Serial number of source nodes
        
         G: Serial number of destination nodes
        
         H: Unit routing cost of links **Real data**
        
         I/J: Unused in this paper
